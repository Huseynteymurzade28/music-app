# Music App

Kısa açıklama: Bu depo iki parçadan oluşur — `backend` (Go) ve `frontend` (Next.js). Aşağıdaki dokümanlar sıfırdan geliştirme ortamı kurmanıza yardımcı olacak.

- **Backend dokümanı:** `backend/README.md`
- **Frontend dokümanı:** `frontend/README.md`

Hızlı başlangıç:

- Backend kurulum adımlarını takip etmek için: `cd backend` → `cat README.md` veya `less README.md`
- Frontend kurulum adımlarını takip etmek için: `cd frontend` → `cat README.md` veya `less README.md`

Her iki klasörde de örnek ortam değişkenleri `*.env.example` olarak yer alır. Kendi `.env` dosyanızı bu örnekten kopyalayın ve değerleri güncelleyin.
